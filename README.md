# Funnier-Doom
A mod that makes the doom mechanic in Slay The Spire 2 funnier by playing funny sound effects on death

You can easily add more sound effects to the existing ones, or replace them alltogether, by simply placing / removing them from the FunnierDoom/FunnyDoomSounds folder.

"Why did you make this mod?"

Because the first human instinct when seeing an enemy being pulled down into hell is to expect a scream. A comedic one, even.
